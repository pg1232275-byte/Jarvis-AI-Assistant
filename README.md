
# Jarvis AI Assistant

Jarvis is a Python-based virtual assistant project developed as part of my journey into automation and artificial intelligence. The objective of this project was not only to build a voice assistant but also to understand how different technologies can be integrated into a single working system.

## What Jarvis Can Do

* Listen to voice commands
* Open websites and applications
* Play music from a predefined library
* Perform web-based tasks
* Interact with AI models for conversational responses
* Execute basic automation workflows

## Why I Built This

After learning Python fundamentals, I wanted to move beyond tutorials and build something practical. Jarvis became my first attempt at combining programming, automation, APIs, and AI into a real project.

During development, I learned how to:

* Structure Python projects
* Work with external APIs
* Handle user input through voice commands
* Debug real-world problems
* Integrate multiple modules into one application

## Project Files

```text
main.py          - Main assistant logic
client.py        - AI model integration basic process
musicLibrary.py  - Music management module
```
